# Dino Panel Flip

GitHub Pages public build for Dino Panel Flip.

## Publish with GitHub Pages

1. Create a public GitHub repository named `dino-panel-flip`.
2. Upload `index.html` from this folder to the repository root.
3. Open repository Settings.
4. Open Pages.
5. Set Source to `Deploy from a branch`.
6. Select `main` and `/root`, then Save.

After GitHub Pages finishes deploying, the game URL will look like:

```text
https://YOUR_GITHUB_USERNAME.github.io/dino-panel-flip/
```
